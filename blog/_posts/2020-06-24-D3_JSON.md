### How to retrieve URL in D3
```js
const url="https://api.spacexdata.com/v2/launchpads";
d3.json(url).then(function(data){
    console.log(data);
};
```
// Promise Pending
const dataPromise = d3.json(url);
console.log("Data Promise: ", dataPromise);
```
